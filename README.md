# 🎰 Slot Machine

A simple console-based Slot Machine game built using Python. This project simulates a classic slot machine where players can place bets, spin the reels, and win payouts based on matching symbols.

## 📖 Overview

The Slot Machine game allows players to:

- View their current balance
- Place bets before each spin
- Spin the slot machine reels
- Win payouts for matching symbols
- Continue playing until they choose to quit

This project was developed to practice Python programming fundamentals, including functions, loops, random number generation, conditional statements, and program organization.

```bash

```

2. Open the project folder.

```bash

```

3. Run the program.

```bash
python slot_machine.py
```

---

## 🎮 Game Flow

1. Start with an initial balance.
2. Enter the amount you want to bet.
3. Spin the slot machine.
4. Match symbols to win payouts.
5. Continue playing until you decide to quit.
